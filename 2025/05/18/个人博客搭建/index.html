五、注意事项
• 本地与线上差异：本地预览时访问量统计会异常（因不蒜子共用统计池），部署后自动恢复

• HTTPS 强制启用：在 GitHub Pages 设置中勾选 Enforce HTTPS

• 自定义域名：在 source 目录创建 CNAME 文件（无后缀）写入域名，并配置 DNS 解析
